### 🏢 ERP Enterprise completo — do estoque à inteligência artificial

O **Bravixo ERP** é uma plataforma de gestão empresarial moderna, construída para escalar operações complexas com controle total sobre vendas, compras, estoque, financeiro e muito mais.

> 🔒 **Repositório privado** — código-fonte protegido. Este README é a vitrine pública do projeto.

</div>

---

## ✨ Módulos do Sistema

### 📊 Principal
- **Dashboard** com visão geral da operação em tempo real

### 🏛️ Admin / Multi-Empresa
- Gestão de **múltiplas empresas** na mesma plataforma
- **Unidades e depósitos** por empresa
- Gestão completa de **usuários**
- **Papéis e permissões** granulares por módulo

### 📋 Cadastros
- **Clientes** e **Fornecedores**
- **Produtos (SKU)** com controle detalhado
- **Parâmetros por depósito**

### 🛒 Vendas
- **Pedidos de venda**
- **Faturamento**
- **Contas a receber**

### 📦 Compras
- **Pedidos de compra**
- **Entradas de mercadoria**
- **Contas a pagar**

### 🏭 Estoque
- **Saldos por depósito**
- **Movimentações** e rastreabilidade
- **Transferências** entre depósitos
- **Ajustes** de inventário

### 💰 Financeiro & Caixa
- **Títulos** financeiros
- **Baixas** e liquidações
- **Conciliação bancária**
- **Sessões de caixa** (abertura/fechamento)
- **Movimentos de caixa**

### 📈 BI — Business Intelligence
- **DRE** — Demonstração de Resultado
- **Fluxo de caixa**
- **Aging** de títulos
- **Exportações** de relatórios

### 🤖 IA — Inteligência Artificial
- Análise **ABC-XYZ** de produtos
- Detecção de **anomalias**
- **Previsão de demanda**
- **Ressuprimento inteligente**

### 🔌 Integrações
- **NF-e** — Emissão e gestão de notas fiscais
- **Pagamentos** integrados

### ⚙️ Sistema
- **Auditoria** completa de ações
- **Configurações** do sistema

## 🏗️ Arquitetura do Projeto

```
Bravixo-ERP/
├── Bravixo-Frontend/       # Next.js + TypeScript
│   └── app/pages/
│       ├── dashboard/      # Visão geral
│       ├── admin/          # Multi-empresa · Usuários · Permissões
│       ├── cadastros/      # Clientes · Fornecedores · Produtos
│       ├── vendas/         # Pedidos · Faturamento · Contas a Receber
│       ├── compras/        # Pedidos · Entradas · Contas a Pagar
│       ├── estoque/        # Saldos · Movimentações · Transferências
│       ├── financeiro/     # Títulos · Baixas · Conciliação
│       ├── caixa/          # Sessões · Movimentos
│       ├── bi/             # DRE · Fluxo de Caixa · Aging
│       ├── ia/             # ABC-XYZ · Anomalias · Previsão
│       ├── integracoes/    # NF-e · Pagamentos
│       └── sistema/        # Auditoria · Configurações
│
├── Bravixo-Api/            # Backend REST
└── Bravixo-Bd/             # Banco de dados PostgreSQL
```

## 🚀 Diferenciais

| Recurso | Descrição |
|---|---|
| 🏢 **Multi-empresa** | Gerencie múltiplas empresas e depósitos em um só sistema |
| 🤖 **IA nativa** | Previsão de demanda, anomalias e ressuprimento inteligente |
| 📊 **BI integrado** | DRE, fluxo de caixa e aging sem ferramentas externas |
| 🔌 **NF-e integrada** | Emissão de notas fiscais direto no sistema |
| 🔐 **Permissões granulares** | Controle total de acesso por papel e módulo |
| 🌐 **100% web** | Acesse de qualquer lugar, sem instalação |
